## 地方型 SBIR vs 中央 SBIR 的差異

### 核心差異

| 項目 | 中央 SBIR | 地方型 SBIR（如台北市） |
|------|----------|----------------------|
| 主辦單位 | 經濟部中小企業處 | 各縣市政府（如台北市產發局） |
| 補助重點 | 技術創新、產業升級 | **地方產業發展、在地就業** |
| 審查標準 | 技術創新性、市場潛力 | **地方連結度、在地效益** |
| 補助金額 | Phase 1: 150萬, Phase 2: 1,200萬 | 依各縣市規定（通常較低） |
| 申請資格 | 全國中小企業 | **需在該縣市設立或營運** |

### 關鍵差異：「地方連結度」

地方型 SBIR 最重視的是：
1. **對地方經濟的貢獻**
2. **在地就業創造**
3. **地方產業鏈帶動**
4. **區域特色發展**

## 成功要素分析

### 要素 1：強調地方連結

**中央 SBIR 寫法**：
```
❌ 「本計畫將開發 AI 客服系統，預計 3 年創造 1,000 萬營收」
```

**地方型 SBIR 寫法**：
```
✅ 「本計畫將在台北市設立研發中心，預計：
- 在台北市創造 10 個就業機會
- 帶動台北市 5 家上下游廠商合作
- 3 年內在台北市創造 1,000 萬營收
- 協助台北市文創產業數位轉型」
```

**差異**：每個數字都要加上「在台北市」或「對台北市」。

### 要素 2：在地就業創造

**必須明確說明**：
- 預計增加多少個工作機會
- 這些工作機會的類型（研發、行銷、製造）
- 薪資水準（證明是「優質就業」）
- 人才培育計畫

**範例**：
```
## 在地就業效益

### 直接就業
- 研發工程師：3 名，月薪 5-7 萬
- 專案經理：2 名，月薪 6-8 萬
- 行銷人員：2 名，月薪 4-5 萬
合計：7 個工作機會

### 間接就業
- 帶動合作廠商增聘：約 5-8 人
- 總計：12-15 個工作機會

### 人才培育
- 與台北市大學合作，提供 5 個實習機會
- 辦理 2 場產業人才培訓課程
```

### 要素 3：地方產業鏈整合

**說明如何帶動地方產業**：

```
## 地方產業鏈整合

### 上游合作
- 台北市 A 公司：提供設計服務
- 台北市 B 工作室：提供內容製作

### 下游合作
- 台北市 C 通路：協助推廣
- 台北市 D 公司：共同開發市場

### 預期效益
- 帶動台北市文創產業鏈 5 家廠商
- 創造產業群聚效應
- 提升台北市文創產業競爭力
```

### 要素 4：區域特色發展

**連結地方產業政策**：

```
## 符合台北市產業政策

本計畫符合「台北市產業發展自治條例」，
響應台北市政府推動的：

1. 文創產業發展計畫
2. 數位轉型政策
3. 智慧城市願景

預期成為台北市文創產業數位化的示範案例。
```

### 要素 5：在地資源運用

**說明如何運用地方資源**：

```
## 在地資源整合

### 場域資源
- 台北市文創園區：作為展示場域
- 台北市創新實驗室：技術驗證

### 人才資源
- 台北市大學：產學合作
- 台北市職訓中心：人才培訓

### 通路資源
- 台北市商圈：試點推廣
- 台北市展覽：品牌曝光
```

## 撰寫架構建議

### 第一章：計畫摘要

**必須包含**：
- 公司在台北市的營運狀況
- 計畫對台北市的貢獻
- 預期創造的在地效益

### 第二章：問題陳述

**重點**：
- 台北市產業面臨的問題
- 本計畫如何解決台北市的問題

### 第三章：創新構想

**一般寫法 + 地方連結**：
- 技術創新（同中央 SBIR）
- **+ 對台北市產業的意義**

### 第四章：市場分析

**重點調整**：
- 台北市市場規模
- 台北市目標客群
- 台北市競爭優勢

### 第五章：執行計畫

**必須說明**：
- 在台北市的執行地點
- 與台北市廠商的合作
- 在台北市的資源運用

### 第六章：團隊組成

**強調**：
- 團隊成員在台北市的經驗
- 對台北市產業的了解

### 第七章：經費規劃

**說明**：
- 經費如何用於台北市
- 對台北市經濟的貢獻

### 第八章：預期成果（最重要！）

**必須量化**：

```
## 預期成果

### 經濟效益
- 3 年在台北市創造營收：1,000 萬
- 帶動台北市產業鏈產值：500 萬

### 就業效益
- 直接就業：7 人
- 間接就業：5-8 人

### 產業效益
- 帶動台北市 5 家廠商合作
- 建立台北市文創產業示範案例

### 社會效益
- 提升台北市文創產業競爭力
- 促進台北市產業數位轉型
```

## 常見錯誤

### 錯誤 1：忽略地方連結

❌ **錯誤**：
> 「本計畫將開發 XX 系統，預計創造 1,000 萬營收」

✅ **正確**：
> 「本計畫將在台北市開發 XX 系統，預計在台北市創造 1,000 萬營收，
> 並在台北市增加 10 個就業機會」

### 錯誤 2：就業效益不明確

❌ **錯誤**：
> 「預計增加就業機會」

✅ **正確**：
> 「預計在台北市增加 7 個直接就業機會：
> - 研發工程師 3 名（月薪 5-7 萬）
> - 專案經理 2 名（月薪 6-8 萬）
> - 行銷人員 2 名（月薪 4-5 萬）」

### 錯誤 3：產業鏈整合空泛

❌ **錯誤**：
> 「將與上下游廠商合作」

✅ **正確**：
> 「將與台北市 5 家廠商合作：
> - A 公司（設計服務）
> - B 工作室（內容製作）
> - C 通路（推廣）
> （已取得合作意向書，見附件）」

## 檢核清單

申請地方型 SBIR 前，請確認：

- [ ] **公司資格**：在該縣市設立或有實質營運
- [ ] **地方連結**：每個數字都標註「在台北市」
- [ ] **就業效益**：明確說明增加多少工作機會
- [ ] **產業鏈**：列出具體合作的在地廠商
- [ ] **區域特色**：連結地方產業政策
- [ ] **在地資源**：說明如何運用地方資源
- [ ] **預期成果**：量化對地方的貢獻
- [ ] **佐證資料**：提供合作意向書、場域證明

## 各縣市特色

### 台北市
- **重點**：文創、數位、服務創新
- **特色**：智慧城市、國際化
- **關鍵字**：數位轉型、創新服務、國際接軌

### 新北市
- **重點**：製造業、傳統產業升級
- **特色**：產業群聚、智慧製造
- **關鍵字**：智慧製造、產業 4.0、循環經濟

### 桃園市
- **重點**：航空、物流、智慧製造
- **特色**：亞洲矽谷、5G 應用
- **關鍵字**：智慧物流、AIoT、5G 創新

### 台中市
- **重點**：精密機械、工具機
- **特色**：智慧機械之都
- **關鍵字**：智慧機械、精密製造、產業升級

### 台南市
- **重點**：綠能、循環經濟
- **特色**：沙崙綠能科學城
- **關鍵字**：綠能科技、循環經濟、智慧農業

### 高雄市
- **重點**：5G、AIoT、智慧城市
- **特色**：亞洲新灣區、5G 應用
- **關鍵字**：5G 應用、智慧港口、循環經濟

## 總結

**地方型 SBIR 的成功公式**：

```
技術創新（50%）
+ 地方連結（30%）
+ 在地就業（10%）
+ 產業鏈整合（10%）
= 高通過率
```

**記住三個關鍵**：
1. **每個數字都要加「在 XX 市」**
2. **就業效益要具體量化**
3. **提供在地廠商合作證明**

**最重要的一句話**：
> 地方型 SBIR 不只是在評估你的技術，
> 更是在評估你對地方經濟的貢獻。
