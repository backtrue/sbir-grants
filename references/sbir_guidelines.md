# SBIR 官方規範與申請指南

> ⚠️ **重要提醒**：本文件整理自公開資訊，規範可能隨時更新，**請務必以經濟部中小及新創企業署官方公告為準**。

## 計畫簡介

**計畫全名**：小型企業創新研發計畫  
**英文名稱**：Small Business Innovation Research (SBIR)  
**主辦機關**：經濟部中小及新創企業署  
**官方網站**：https://www.sbir.org.tw/

### 計畫宗旨

SBIR 計畫旨在：
1. 鼓勵國內中小企業加強技術與服務創新研發
2. 提升中小企業創新研發能力與成果商品化價值
3. 引導學研機構協助中小企業強化研發能量
4. 帶動國內產業升級轉型與經濟發展

---

## 2026年度重要更新

自 2026 年 1 月 1 日起，SBIR 計畫有以下重大調整：

### 補助金額調高

| 階段 | 舊上限 | **新上限** |
|------|--------|------------|
| Phase 1 先期研究 | 100萬元 | **150萬元** |
| Phase 2 研究開發 | 1,000萬元 | **1,200萬元** |
| Phase 2+ 加值應用 | 500萬元 | **600萬元** |

### 新增補助項目

- **專利申請費**：支持企業在研發初期布局國內外專利

### 重點領域

2026 年度鎖定三大核心領域：
1. **無人載具應用**
2. **AI 智慧應用**
3. **淨零應用**

### 主題式徵案

導入「主題式徵案」機制，促成中小企業跨領域技術結盟。

## 申請資格

### 基本條件

符合「中小企業認定標準」之台灣企業：

**資本額標準**（二擇一）：
- 實收資本額在新台幣 **1 億元以下**，或
- 經常僱用員工數 **未滿 200 人**

**企業類型**：
- 依公司法設立之公司
- 依有限合夥法設立之有限合夥
- 依商業登記法設立之商業

### 排除條件

以下情況 **不得申請**：

| 排除項目 | 說明 |
|----------|------|
| 陸資企業 | 有大陸地區人民、法人、團體或其他機構直接或間接投資 |
| 外商分公司 | 外國營利事業在台設立之分公司或分支機構 |
| 重大違約 | 近 5 年內有政府計畫重大違約紀錄 |
| 欠稅紀錄 | 近 3 年內有欠稅情事 |
| 法規違反 | 嚴重違反環保、勞工、食安等相關法規 |
| 信用不良 | 銀行拒絕往來戶 |
| 淨值為負 | 最近一年度企業淨值為負數 |

### 同時執行限制

- 每家企業同時執行中的 SBIR 計畫以 **2 案** 為限
- 申請第 2 案，需前案審查評分達 **88 分以上**
- 已獲其他政府創新研發計畫補助且尚未結案者，不得重複申請

## 計畫類型

### 中央型 SBIR（一般型計畫）

**特色**：隨到隨審，無申請期限

| 階段 | 補助上限 | 期程 | 目標 |
|------|----------|------|------|
| Phase 1 | 150萬元 | 6個月 | 創新構想可行性驗證 |
| Phase 2 | 1,200萬元 | 6-24個月 | 產品雛形開發、小量試產 |
| Phase 2+ | 600萬元 | 依計畫 | 商業化、規模化 |

**補助比例**：最高 50%（需自籌 50%）

### 地方型 SBIR

**特色**：配合地方政府推動，依地方產業特色制定主題

| 項目 | 內容 |
|------|------|
| 補助上限 | 約 100萬元 |
| 主題 | 依各縣市產業特色訂定 |
| 受理方式 | 依各地方政府公告 |

### 創業海選

**特色**：針對創業初期團隊

| 項目 | 內容 |
|------|------|
| 補助金額 | 20萬元 或 60萬元 |
| 對象 | 創業初期企業 |

## 申請流程

### 中央型 SBIR 申請步驟

```
1. 確認資格 → 2. 準備文件 → 3. 線上申請 → 4. 形式審查 → 5. 實質審查 → 6. 核定公告
```

#### 步驟說明

1. **確認資格**
   - 確認符合中小企業認定標準
   - 確認無排除條件
   - 確認執行案件數量限制

2. **準備文件**
   - 公司登記證明文件
   - 最近一年度財務報表
   - 計畫書（或簡報，Phase 1）
   - 其他佐證資料

3. **線上申請**
   - 至 SBIR 官方網站註冊帳號
   - 填寫線上申請表
   - 上傳計畫書及相關文件

4. **形式審查**
   - 確認文件齊全
   - 確認資格符合
   - 約 1-2 週

5. **實質審查**
   - Phase 1：簡報審查
   - Phase 2：書面審查 + 簡報審查
   - 約 1-2 個月

6. **核定公告**
   - 審查通過後公告核定
   - 簽訂補助契約
   - 開始執行計畫

## 經費規範

### 可補助項目

| 類別 | 項目 | 說明 |
|------|------|------|
| **人事費** | 研發人員薪資 | 參與計畫之人員薪資 |
| **耗材費** | 消耗性器材及原材料 | 實驗材料、軟體授權等 |
| **設備使用費** | 研發設備使用費 | 設備租賃或折舊分攤 |
| **技術引進費** | 技術授權費 | 技術移轉或授權費用 |
| **委託研究費** | 外包研發費 | 委託外部單位進行研發 |
| **差旅費** | 國內外差旅 | 研發相關出差費用 |
| **專利申請費** | 專利相關費用 | 國內外專利申請費（2026年新增） |
| **其他費用** | 其他必要支出 | 文獻、資料蒐集等 |

### 經費限制

| 項目 | 限制 |
|------|------|
| 人事費 | 不超過總經費 60% |
| 委託研究費 | 不超過總經費 30% |
| 補助比例 | 最高 50%（需自籌 50%） |

### 不可補助項目

- ❌ 一般行政管理費用
- ❌ 非計畫相關人員薪資
- ❌ 非研發用途之設備
- ❌ 土地及建築物購置
- ❌ 營業稅（可外加報銷）

## 審查機制

### Phase 1 審查

**方式**：簡報審查

**流程**：
1. 上傳申請資料
2. 安排簡報審查時間
3. 現場簡報 15-20 分鐘
4. 委員提問 10-15 分鐘
5. 評分與決議

**評審重點**：
- 創新構想的創新性
- 問題定義的明確性
- 可行性評估的合理性
- 團隊執行能力

### Phase 2 審查

**方式**：書面審查 + 簡報審查

**流程**：
1. 書面審查（初審）
2. 通過者進入簡報審查
3. 現場簡報 20-30 分鐘
4. 委員提問 15-20 分鐘
5. 評分與決議

**評審重點**：
- 技術創新性與可行性
- 市場分析與商業化規劃
- 團隊能力與執行計畫
- 經費編列合理性
- 預期產業效益

## 計畫執行

### 執行規範

1. **專款專用**
   - 補助款須專戶管理
   - 依核定項目支用

2. **定期報告**
   - 期中報告（視計畫期程）
   - 結案報告

3. **查核機制**
   - 不定期訪視查核
   - 確認執行進度

4. **變更程序**
   - 計畫內容變更需申請
   - 經費流用有比例限制

### 結案要求

1. **成果報告**
   - 完成結案報告書
   - 說明成果與效益

2. **經費核銷**
   - 提供支出憑證
   - 核銷補助經費

3. **績效追蹤**
   - 結案後持續追蹤效益
   - 填報績效調查表
