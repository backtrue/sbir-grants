# 智財策略指南

> 研發成果要轉化為競爭優勢，智財佈局是關鍵。

## 為什麼智財重要？

### 對 SBIR 計畫的影響

| 面向 | 說明 |
|------|------|
| 審查加分 | 有專利規劃展示技術獨特性 |
| 成果保護 | 避免研發成果被抄襲 |
| 商業價值 | 專利可授權或交易 |
| 2025 新政策 | SBIR 新增專利費用補助 |

### 智財類型比較

| 類型 | 保護對象 | 保護期限 | 申請難度 | 成本 |
|------|----------|----------|----------|------|
| 發明專利 | 新技術、方法 | 20 年 | 高 | 高 |
| 新型專利 | 物品構造 | 10 年 | 低 | 中 |
| 設計專利 | 外觀設計 | 15 年 | 中 | 中 |
| 營業秘密 | know-how | 無期限 | - | 低 |
| 著作權 | 軟體、文件 | 終身+50年 | - | 低 |

## 專利 vs 營業秘密

### 何時選擇專利？

✅ **適合申請專利的情況**：
- 技術容易被逆向工程
- 需要公開技術換取保護
- 希望阻止競爭者
- 計畫授權或出售技術

### 何時選擇營業秘密？

✅ **適合營業秘密的情況**：
- 製程配方難以被逆向
- 不想公開技術細節
- 技術生命週期短（< 3 年）
- 申請專利成本太高

### 比較表

| 項目 | 專利 | 營業秘密 |
|------|------|----------|
| 公開性 | 必須公開 | 保密 |
| 保護期限 | 有期限 | 無期限 |
| 保護範圍 | 申請國家 | 全球（如能保密） |
| 取得成本 | 較高 | 較低 |
| 維持成本 | 年費 | 管理成本 |
| 他人獨立發明 | 可主張權利 | 無法阻止 |

## 專利申請流程

### 台灣發明專利流程

```
構想 → 檢索 → 撰寫 → 送件 → 審查 → 核准 → 公告
  ↓      ↓      ↓      ↓      ↓      ↓      ↓
 1週   2-4週  2-4週   -    18-24月  -     1月
```

**時程參考**：
- 新型專利：約 4-6 個月
- 發明專利：約 18-24 個月
- 可申請加速審查

### 專利申請成本

| 項目 | 發明專利 | 新型專利 |
|------|----------|----------|
| 規費 | 約 6,000 | 約 3,000 |
| 代理人費用 | 3-8 萬 | 1.5-3 萬 |
| 年費（前 3 年） | 約 7,500 | 約 4,500 |

**2025 新政策**：SBIR 計畫可補助專利申請費用！

## 專利檢索

### 為什麼要做專利檢索？

1. **避免侵權**：確認技術不侵害他人專利
2. **評估新穎性**：確認技術有專利價值
3. **了解競爭態勢**：知道誰在這領域佈局
4. **找技術參考**：學習現有技術

### 免費檢索工具

| 工具 | 網址 | 涵蓋範圍 |
|------|------|----------|
| 智慧局全球專利檢索 | gpss.tipo.gov.tw | 台灣 + 全球 |
| Google Patents | patents.google.com | 全球 |
| USPTO | www.uspto.gov | 美國 |
| EPO Espacenet | worldwide.espacenet.com | 歐洲 + 全球 |
| WIPO | www.wipo.int/patentscope | PCT |

### 檢索技巧

1. **關鍵字檢索**
   - 使用多個同義詞
   - 組合關鍵字（AND、OR）
   - 使用萬用字元

2. **分類號檢索**
   - 找到相關 IPC/CPC 分類
   - 更精準找到相關專利

3. **引證檢索**
   - 從已知專利找相關專利
   - 追蹤技術發展脈絡

## 計畫書中的智財呈現

### 在計畫書中加分的寫法

#### 1. 專利佈局規劃

```
智財策略規劃：

1. 已有專利：
   - 發明專利 TW I123456「XX 方法及系統」（2023 年核准）
   
2. 計畫中擬申請：
   - 發明專利 1 件：核心演算法（Phase 1 期間申請）
   - 新型專利 2 件：機構設計（Phase 2 期間申請）
   
3. 營業秘密保護：
   - 製程參數（內部管理）
   - 客戶數據（保密協議）
```

#### 2. 技術迴避設計

```
本計畫技術與現有專利的差異：

現有專利：US 9,123,456（XX 公司）
- 採用方法 A
- 使用材料 X
- 準確率 85%

本計畫技術：
- 採用創新方法 B（迴避現有專利）
- 使用新型材料 Y
- 準確率 92%

結論：本計畫技術與現有專利有明確差異，無侵權風險。
```

### 審查委員會看的重點

| 項目 | 評審會想 |
|------|----------|
| 新穎性 | 「這技術真的是新的嗎？」 |
| 可專利性 | 「有專利價值嗎？」 |
| 侵權風險 | 「會不會侵害他人專利？」 |
| 保護策略 | 「有沒有保護計畫？」 |

## 產學合作的智財處理

### 智財歸屬原則

| 情境 | 通常歸屬 |
|------|----------|
| 公司獨立研發 | 公司所有 |
| 學校獨立研發 | 學校所有 |
| 共同研發 | 共同所有 |
| 委託研究 | 依合約約定 |

### 合約注意事項

**必須約定清楚**：
1. 前景智財（計畫產出）歸誰？
2. 背景智財（既有技術）如何處理？
3. 改良發明歸誰？
4. 商業化權利如何分配？
5. 授權金如何分配？

## 專利申請時機

### 最佳申請時機

```
     研發開始    雛型完成    產品上市
        ↓           ↓           ↓
────────●───────────●───────────●──────────
        │           │           │
        │           │           └─ 太晚！可能已被公開
        │           │
        │           └─ 理想時機：技術成熟但未公開
        │
        └─ 可以：基礎概念先申請
```

### 注意：公開會影響新穎性

**以下情況會喪失新穎性**：
- 論文發表
- 展覽展示
- 網站公開
- 新聞報導
- 未簽 NDA 的討論

**台灣有 6 個月優惠期**，但建議仍在公開前申請。

## 常見問題 FAQ

### Q1：還沒有產品，可以申請專利嗎？
**A**：可以。只要有「可實施的技術方案」就可以申請。不需要有實際產品。

### Q2：申請專利一定要找代理人嗎？
**A**：不一定，但強烈建議。專利撰寫有特殊技巧，自己寫容易保護範圍太窄。

### Q3：專利申請中可以公開技術嗎？
**A**：可以，但建議等「公開後」再公開。發明專利申請後 18 個月會公開。

### Q4：多國申請要花很多錢嗎？
**A**：是的。每個國家都要費用。建議評估市場後再決定申請哪些國家。

### Q5：軟體可以申請專利嗎？
**A**：可以，但要以「技術手段解決技術問題」的方式撰寫，不是純商業方法。

## 智財管理建議

### 日常管理

1. **發明記錄**：研發過程要留記錄
2. **保密管理**：重要文件要分級管理
3. **員工規範**：簽署保密與發明歸屬協議
4. **定期檢視**：每季檢視智財現況

### 專利組合策略

```
核心專利：保護關鍵技術
    ↓
週邊專利：保護延伸應用
    ↓
防禦專利：阻止競爭者接近
```
