# 商業化策略規劃

## 概述

SBIR 計畫不只是做研發，更重要的是能不能商業化。審查委員會問：「做出來之後呢？」本指南協助你規劃有說服力的商業化策略。

---

## 商業模式設計

### 價值主張

清楚說明你提供的價值：

**撰寫方式**：
```
我們幫助 [目標客戶]
解決 [具體問題]
透過 [你的解決方案]
讓他們能夠 [獲得的效益]
```

**範例**：
```
我們幫助中小型製造業
解決庫存預測不準確的問題
透過 AI 智慧庫存管理系統
讓他們能夠降低庫存成本 30%、減少缺貨率 50%
```

### 營收模式

常見的營收模式：

| 模式 | 說明 | 適用情境 |
|------|------|----------|
| 訂閱制 | 按月/年收費 | SaaS 軟體 |
| 授權費 | 一次性授權 | 軟體、專利 |
| 銷售收入 | 產品銷售 | 硬體、設備 |
| 服務費 | 導入/顧問服務 | 系統整合 |
| 抽成/佣金 | 交易抽成 | 平台服務 |
| 維護費 | 年度維護 | 軟硬體 |

### 定價策略

**定價方法**：
1. 成本加成法：成本 + 合理利潤
2. 競爭定價法：參考競品定價
3. 價值定價法：根據客戶獲得的價值

**定價說明範例**：
```
定價策略：訂閱制 + 導入服務費

- 月費：2 萬/月（含 10 個用戶）
- 年費：20 萬/年（8.3 折）
- 導入費：10-30 萬（依客製程度）
- 維護費：年費的 15%

定價基礎：競品月費約 3-5 萬，
我們以較低定價搶攻中小企業市場。
預估毛利率 60%。
```

## 通路策略

### 銷售通路

| 通路 | 優點 | 缺點 |
|------|------|------|
| 直銷 | 利潤高、控制強 | 成本高、規模慢 |
| 經銷商 | 擴展快、在地化 | 利潤低、控制弱 |
| 線上平台 | 成本低、規模大 | 競爭激烈 |
| OEM/ODM | 量大、穩定 | 毛利低 |

### 通路規劃範例

```
Year 1: 直銷為主
- 自建業務團隊 2 人
- 聚焦大台北地區

Year 2: 直銷 + 合作夥伴
- 與 3 家系統整合商合作
- 擴展至北中南

Year 3: 多元通路
- 增加線上銷售管道
- 開拓海外經銷
```

## 市場進入策略

### 階段性規劃

```
Phase 1: 種子客戶期
- 目標：取得 3-5 家指標客戶
- 策略：免費/優惠試用
- 重點：建立案例、收集回饋

Phase 2: 市場建立期
- 目標：擴展至 20-30 家客戶
- 策略：案例行銷、口碑推薦
- 重點：優化產品、建立口碑

Phase 3: 規模成長期
- 目標：100 家以上客戶
- 策略：通路擴展、品牌行銷
- 重點：規模化、提高效率
```

### 客戶開發策略

**開發方式**：
1. 目標企業名單建立
2. 主動接觸/展會參展
3. 免費試用/POC
4. 成案合作
5. 滿意度追蹤


## 財務預估

### 營收預估

| 年度 | 客戶數 | 客單價 | 營收 |
|------|--------|--------|------|
| Y1 | 5 | 30萬 | 150萬 |
| Y2 | 20 | 30萬 | 600萬 |
| Y3 | 50 | 35萬 | 1,750萬 |

### 損益預估

| 項目 | Y1 | Y2 | Y3 |
|------|-----|-----|-----|
| 營收 | 150萬 | 600萬 | 1,750萬 |
| 成本 | 60萬 | 200萬 | 550萬 |
| 毛利 | 90萬 | 400萬 | 1,200萬 |
| 費用 | 200萬 | 350萬 | 600萬 |
| 淨利 | -110萬 | 50萬 | 600萬 |

### 損益平衡分析

```
固定成本：200 萬/年
變動成本率：40%
客單價：30 萬

損益平衡客戶數 = 200萬 / (30萬 × 60%) = 11 家
預估於營運第 2 年達到損益平衡
```

## IP 策略

### 智慧財產保護

| 方式 | 適用 | 說明 |
|------|------|------|
| 發明專利 | 技術方法 | 保護期 20 年 |
| 新型專利 | 結構設計 | 保護期 10 年 |
| 著作權 | 軟體程式 | 自動保護 |
| 營業秘密 | Know-how | 保密管理 |
| 商標 | 品牌名稱 | 需註冊 |

### IP 規劃範例

```
本計畫 IP 策略：
- 核心演算法申請發明專利 2 件
- 軟體程式進行著作權登記
- 技術文件列為營業秘密
- 品牌商標已完成註冊
```

## 常見錯誤

1. **商業模式不清**：只說要賣，沒說怎麼賣
2. **定價沒根據**：定價沒有計算基礎
3. **營收預估誇大**：不切實際的成長預期
4. **通路策略模糊**：不知道怎麼接觸客戶
5. **忽略 IP 保護**：沒有智財保護規劃
