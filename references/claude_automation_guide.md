## 🤖 Claude 自動化指引

> **給 Claude**：當用戶需要市場數據時，請按照以下流程自動執行。

---

### 觸發情境

當用戶提到以下關鍵字時，**自動啟動數據查詢流程**：

- 「我要寫 [產業] 的問題陳述」
- 「幫我找 [產業] 的市場數據」
- 「需要 [產業] 的產業報告」
- 「查詢 [產業] 市場規模」

---

### 自動執行流程（混合式）

#### Step 1: 呼叫 MCP Server（經濟部統計處）

```
使用工具：query_moea_statistics
參數：
  - industry: [用戶提到的產業]
  - stat_type: "產值" 或 "出口"
  - start_year: 2020
  - end_year: 2024
```

#### Step 2: 使用 search_web（工研院 IEK）

```
search_web("[產業] [關鍵字] 2024 site:iek.itri.org.tw")

範例：
search_web("機械產業 市場規模 2024 site:iek.itri.org.tw")
```

#### Step 3: 使用 search_web（資策會 MIC）

**僅當產業為「資通訊」或「電子」時執行**：

```
search_web("[產業] [關鍵字] 2024 site:mic.iii.org.tw")

範例：
search_web("AI應用 市場規模 2024 site:mic.iii.org.tw")
```

#### Step 4: 整合結果

將三個來源的數據整理成以下格式：

```markdown
## [產業]產業市場數據

### 1. 經濟部統計處（官方數據）

根據經濟部統計處 (2024)，台灣[產業]產業...

**來源**：https://www.moea.gov.tw/Mns/dos/

### 2. 工研院 IEK（產業報告）

根據工研院 IEK (2024) 報告，...

**來源**：https://iek.itri.org.tw/

### 3. 資策會 MIC（資通訊數據）

根據資策會 MIC (2024)，...

**來源**：https://mic.iii.org.tw/

---

## 可引用段落

您可以這樣寫入計畫書：

> 根據工研院 IEK (2024) 報告，台灣[產業]產業市場規模達 XX 億元，
> 2024-2030 年複合成長率 (CAGR) 預估為 XX%。
> 
> 來源：https://iek.itri.org.tw/
```

### 搜尋技巧

#### 產業別對應的搜尋關鍵字

| 產業 | 推薦關鍵字 | 優先來源 |
|------|-----------|----------|
| 機械 | 工具機、智慧製造、產值 | 工研院 IEK、經濟部統計處 |
| 生技 | 生技醫藥、醫療器材、市場規模 | 工研院 IEK、生技中心 |
| 資通訊 | AI應用、雲端服務、ICT產業 | 資策會 MIC、工研院 IEK |
| 化工 | 化學材料、循環經濟、產值 | 工研院 IEK、經濟部統計處 |
| 電子 | 半導體、電子元件、產值 | 工研院 IEK、資策會 MIC |
| 服務業 | 電商、零售、營業額 | 經濟部統計處、商研院 |

#### 提高搜尋精準度

1. **加上年份**：`機械產業 市場規模 2024`
2. **限定網站**：`site:iek.itri.org.tw`
3. **使用引號**：`"台灣機械產業"` （精確匹配）
4. **排除無關**：`-新聞 -廣告`

### 錯誤處理

#### 情況 1：MCP Server 查詢失敗

**處理方式**：
- 改用 `search_web` 查詢經濟部統計處網站
- 告知用戶：「經濟部 API 暫時無法使用，已改用網站搜尋」

#### 情況 2：找不到數據

**處理方式**：
1. 調整關鍵字（如：「市場規模」→「產值」）
2. 擴大年份範圍（如：2024 → 2020-2024）
3. 建議用戶：「建議調整關鍵字或查詢更廣泛的產業類別」

#### 情況 3：數據過舊

**處理方式**：
- 註明數據年份
- 使用 CAGR 推估至目標年度
- 範例：「根據 2022 年數據，以 CAGR 5% 推估，2024 年約為 XX 億元」

**用戶**：「我要寫機械產業的問題陳述，請幫我找市場數據」

### 注意事項

1. **優先使用官方數據**：經濟部統計處 > 工研院 IEK > 資策會 MIC
2. **註明來源與年份**：每個數據都要標註來源網址與年份
3. **避免過度依賴單一來源**：至少引用 2 個來源
4. **數據一致性檢查**：如果不同來源數據差異過大，提醒用戶確認
