# 市場分析撰寫指南

## 概述

市場分析是 SBIR 計畫書與 NSTC 最大的差異之一。SBIR 審查委員特別關注：你的產品/服務能不能賣？能賣多少？市場分析就是要回答這些問題。

---

## TAM/SAM/SOM 框架

這是最常用的市場規模分析框架：

### TAM (Total Addressable Market)

**定義**：如果你佔據 100% 市場，能有多大？

**範例**：全球製造業 ERP 軟體市場 = 500 億美元

### SAM (Serviceable Available Market)

**定義**：以你目前的產品/服務能觸及的市場

**範例**：亞太區中型製造業 ERP 市場 = 50 億美元

### SOM (Serviceable Obtainable Market)

**定義**：你實際能獲得的市場份額

**範例**：台灣中小製造業（首年目標）= 2 億台幣

### 呈現方式

| 層級 | 市場定義 | 規模 | 計算基礎 |
|------|----------|------|----------|
| TAM | 全球製造業ERP市場 | 500億美元 | Gartner報告 |
| SAM | 亞太區中型製造業 | 50億美元 | IDC報告 |
| SOM | 台灣中小製造業(Y1) | 2億台幣 | 目標客戶數×客單價 |

1. **公開報告摘要**：研究機構常公開部分數據

3. **產業新聞**：追蹤產業媒體報導
4. **自行估算**：根據既有數據推算

## 競爭分析

### 競爭者識別

列出主要競爭者：
- 直接競爭者（同類產品）
- 間接競爭者（替代方案）
- 潛在競爭者（可能進入者）

### 競爭比較表

| 項目 | 競爭者A | 競爭者B | **本公司** |
|------|---------|---------|------------|
| 產品名稱 | 產品X | 產品Y | **產品Z** |
| 定價 | 高 | 中 | **中低** |
| 功能完整度 | ★★★★★ | ★★★☆☆ | **★★★★☆** |
| 在地化 | ★★☆☆☆ | ★★★☆☆ | **★★★★★** |
| 技術支援 | ★★★☆☆ | ★★★★☆ | **★★★★★** |

### 競爭優勢論述

**差異化來源**：
1. 技術優勢：有什麼獨特技術？
2. 成本優勢：為什麼能更便宜？
3. 服務優勢：有什麼更好的服務？
4. 在地優勢：本土化的優勢？

## 目標客戶分析

### 客戶輪廓

描述你的目標客戶：
- **產業**：哪些產業？
- **規模**：多大的企業？
- **地區**：台灣？海外？
- **特徵**：有什麼共同特徵？

**範例**：
```
目標客戶：台灣中小型製造業
- 員工人數：50-500人
- 年營收：1-10億台幣
- 產業：電子、金屬加工、塑膠
- 痛點：庫存管理效率低、人工作業多
```

### 客戶痛點

列出客戶面臨的問題：
1. 痛點一：描述問題
2. 痛點二：描述問題
3. 痛點三：描述問題

### 客戶需求驗證

**驗證方式**：
- 客戶訪談（至少 5-10 家）
- 問卷調查
- POC/試用
- LOI（合作意向書）

**呈現方式**：
```
我們訪談了 15 家潛在客戶：
- 12 家表示有此痛點
- 8 家表示願意試用
- 3 家已提供 LOI
```

## 市場驅動因素

說明為什麼這個市場會成長：

### 正向驅動

| 因素 | 說明 |
|------|------|
| 政策推動 | 政府推動數位轉型 |
| 技術成熟 | AI技術普及化 |
| 成本下降 | 雲端服務成本降低 |
| 需求增加 | 企業數位化需求增加 |

### 成長預測

引用權威預測：
```
根據 Gartner 預測，台灣製造業 AI 應用市場
將從 2025 年的 50 億成長到 2028 年的 150 億，
CAGR 達 45%。
```

## 常見錯誤

### 1. 市場規模誇大

**問題**：用全球最大市場數字
**對策**：聚焦到 SOM，說明計算基礎

### 2. 數據來源不明

**問題**：「市場很大」但沒有引用來源
**對策**：標註所有數據來源

### 3. 競爭分析不足

**問題**：忽略競爭者或只說「沒有競爭者」
**對策**：誠實分析，說明差異化

### 4. 沒有客戶驗證

**問題**：只有假設沒有驗證
**對策**：進行客戶訪談，取得 LOI

## 撰寫範例

### 市場分析段落範例

```
根據 MIC 研究，台灣製造業庫存管理軟體市場規模
於 2025 年達 80 億台幣，預估 2028 年將成長至
120 億，CAGR 約 15%。

本計畫聚焦於員工 50-500 人的中型製造業，
此區間約佔整體市場的 40%（約 32 億）。
以首年導入 10 家客戶、平均客單價 200 萬計算，
首年目標營收為 2,000 萬台幣，佔 SOM 約 0.6%。

透過前期訪談 15 家目標客戶，80% 表示現有
庫存管理方式效率低落，平均每月花費 40 小時
進行人工盤點。其中 3 家已提供合作意向書，
待產品上市後優先導入。
```
