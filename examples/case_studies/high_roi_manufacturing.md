# 高 ROI 成功案例：智慧化生產系統

## 案例概述

**公司背景**：
- 公司名稱：（匿名）台灣機械零件製造公司
- 產業：機械零件製造
- 員工數：50 人
- 年營收：8,000 萬

**計畫名稱**：智慧化生產排程與品質監控系統

**補助階段**：Phase 1
**補助金額**：150 萬元
**計畫期程**：12 個月（2024/01 - 2024/12）

---

## 計畫內容

### 問題陳述

傳統機械零件製造面臨：
1. **生產排程效率低**：人工排程，經常延誤交期
2. **品質不穩定**：良率僅 85%，不良品損失大
3. **人力成本高**：需要 3 名專職品檢人員

**量化痛點**：
- 每年因延誤交期損失訂單：500 萬
- 每年不良品損失：150 萬
- 品檢人力成本：180 萬/年

### 創新構想

開發「AI 驅動的智慧生產系統」：
1. **智慧排程**：AI 自動優化生產排程
2. **即時品檢**：機器視覺自動檢測
3. **預測維護**：預測設備故障，減少停機

**技術創新點**：
- 整合 IoT 感測器 + 機器視覺 + AI 演算法
- 台灣首創針對中小型機械廠的低成本解決方案

---

## 預期成果（計畫書中的撰寫）

### 直接產值（營收）

**市場分析**：
- 台灣機械零件製造業：500 億/年
- 中小型機械廠：約 200 家
- 目標客群：年營收 5,000 萬 - 2 億的機械廠（約 80 家）

**產品定位**：
- 價格：40 萬/套（含硬體 + 軟體 + 1 年維護）
- 年度維護費：8 萬/年

**營收預估**：

| 年度 | 客戶數 | 銷售收入 | 維護收入 | 合計 |
|------|--------|---------|---------|------|
| 第 1 年 | 2 家 | 80 萬 | 0 | 80 萬 |
| 第 2 年 | 6 家 | 240 萬 | 16 萬 | 256 萬 |
| 第 3 年 | 12 家 | 480 萬 | 64 萬 | 544 萬 |
| **合計** | **20 家** | **800 萬** | **80 萬** | **880 萬** |

**計算依據**：
1. **市場規模**：根據工研院產業報告
2. **客戶數量**：
   - 已有 3 家簽署合作意向書（附件）
   - 完成 10 家客戶訪談，8 家表示有興趣
   - 保守估計轉換率 30%
3. **定價**：
   - 參考國外類似系統（100-200 萬）
   - 我們定價 40 萬（低價策略）
   - 客戶願付價格調查：平均 45 萬
4. **成長曲線**：
   - 第 1 年：試銷 + 案例累積
   - 第 2 年：口碑擴散
   - 第 3 年：穩定成長

### 間接產值

**自身工廠導入效益**：

| 項目 | 改善幅度 | 年度效益 | 3 年累積 |
|------|---------|---------|---------|
| 生產效率提升 | +25% | 100 萬 | 300 萬 |
| 良率提升 | 85% → 95% | 50 萬 | 150 萬 |
| 減少品檢人力 | -2 人 | 120 萬 | 360 萬 |
| **合計** | - | **270 萬/年** | **810 萬** |

**計算依據**：
- 試驗場域驗證數據（6 個月試運行）
- 生產效率：實測提升 28%，保守取 25%
- 良率：實測提升至 96%，保守取 95%

### 總產值與 ROAS

```
直接產值（營收）：880 萬
間接產值（成本節省）：810 萬
總產值：1,690 萬

ROAS = 1,690 / 150 = 11.3 倍 ⭐⭐⭐
```

**保守估計**（僅計直接產值）：
```
ROAS = 880 / 150 = 5.9 倍 ⭐
```

## 成功關鍵因素

### 1. 精準的市場定位

**關鍵決策**：
- ❌ 不做大廠（已有預算買國外系統）
- ✅ 專攻中小廠（價格敏感，需要低成本方案）

**結果**：
- 避開紅海競爭
- 找到明確的利基市場

### 2. 保守但有依據的產值預估

**策略**：
```
理論市佔率：15%（12 家 / 80 家）
保守估計：取 50% = 7.5%（6 家）
實際達成：10%（8 家）

「寧可低估後超標，也不要高估後失望」
```

### 3. 充分的客戶驗證

**做法**：
- 計畫書撰寫前：完成 10 家客戶訪談
- 取得 3 家合作意向書（附在計畫書中）
- 在自家工廠試運行 6 個月，有實測數據

**效果**：
- 審查委員高度肯定「有充分市場驗證」
- 評分：創新性 85 分、市場性 90 分、可行性 88 分

### 4. 分階段執行策略

**Phase 1 策略**：
- 目標：技術驗證 + 2-3 個標竿客戶
- 不求快速擴張，求穩定品質

**Phase 2 規劃**（已獲補助 800 萬）：
- 目標：產品化 + 通路建立
- 預計 3 年累積營收 5,000 萬

## 給申請者的啟示

### 啟示 1：產值要「合理樂觀」

**錯誤做法**：
```
❌ 過度保守：補助 150 萬，產值寫 200 萬
   → 審查委員：「沒價值」

❌ 過度樂觀：補助 150 萬，產值寫 5,000 萬
   → 審查委員：「不可信」
```

**正確做法**：
```
✅ 合理樂觀：補助 150 萬，產值寫 880 萬（5.9 倍）
   → 有依據、有驗證、有保守估計
```

### 啟示 2：數字一定要有依據

**本案例的依據**：
1. 市場規模：工研院報告
2. 客戶數量：意向書 + 訪談
3. 定價：市場調查 + 競品分析
4. 效益：試驗場域實測數據

**審查委員最常問的問題**：
> 「你的營收預估怎麼來的？」

**標準答案**：
> 「市場規模 X 億，我們目標市佔率 Y%，
> 對應營收 Z 萬。已有 N 家客戶意向書，
> 並完成 M 家試用驗證。」

### 啟示 3：用「保守估計」建立信任

**策略**：
```
理論值：1,500 萬
保守估計：取 60% = 900 萬
實際目標：800 萬

說明：「雖然理論上可達 1,500 萬，
但考慮執行風險，我們保守估計為 900 萬，
實際目標設定為 800 萬」
```

**效果**：
- 審查委員覺得「務實、可信」
- 即使打折，ROAS 仍達標

## 總結

**本案例的成功公式**：

```
精準市場定位
+ 充分客戶驗證
+ 保守但有依據的產值預估
+ 分階段執行策略
= 高 ROI + 高通過率
```

**最重要的一句話**：

> 「SBIR 不是在考你會不會吹牛，
> 而是在考你有沒有想清楚商業模式。」

**記住**：
- 補助 1 元 → 產值至少 3 元（最低標準）
- 補助 1 元 → 產值 5-10 元（建議目標）
- 每個數字都要有依據
- 合理樂觀，不誇大
